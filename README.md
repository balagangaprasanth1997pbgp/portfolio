## Hosting on GitHub Pages

This project is completely Vanilla HTML, CSS, and JavaScript and requires zero build steps or npm configuration.

To host this on GitHub Pages:
1. Create a new repository on GitHub.
2. Upload all the files (`index.html`, `style.css`, `script.js`, and the `assets` folder) to the main branch.
3. Go to your repository's **Settings**.
4. Scroll down to the **Pages** section on the left sidebar.
5. Under **Build and deployment**, select **Deploy from a branch**.
6. Select the `main` branch and `/root` folder, then click **Save**.
7. Wait a couple of minutes, and your site will be live!
